 Dual-Mode AI Chatbot: Academic & Friendly Assistant

An intelligent AI chatbot designed to pivot between structured academic tutoring and casual, friendly conversation. Built as a portfolio project for CSE AIML engineering.

 🚀 Core Features
* **Dual-Response Architecture**: Switches between two distinct conversational personas.
  * **Academic Mode**: Delivers structured technical explanations, step-by-step breakdowns, and formal guidance.
  * **Friendly Mode**: Employs casual phrasing, empathy, and lighthearted conversation.
* **Streamlined Design**: Built entirely within a lightweight, two-file structure for fast deployment.

 📁 Repository Structure
* `app.py` — Contains the frontend interface, core chatbot engine, and prompt engineering logic.
* `requirements.txt` — Lists all Python dependency declarations needed to run the application.

 ⚡ Quick Start

 1. Clone the Repository
```bash
git clone https://github.com
cd YOUR_REPOSITORY_NAME
```

 2. Install Dependencies
```bash
pip install -r requirements.txt
```

 3. Run the Application
```bash
python app.py
```

 🧠 AIML Focus
This project utilizes **System Prompt Engineering** to establish distinct behavioral boundaries within the language model backend, demonstrating clean text manipulation and contextual persona control.
